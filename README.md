## 🛠️ Tech Stack  
**Frontend**: React.js, Tailwind CSS, Socket.io  
**Backend**: Node.js, Express, Firebase Firestore  
**AI**: OpenAI GPT-3.5 Turbo API  
**Deployment**: Vercel (Frontend), Render (Backend)
